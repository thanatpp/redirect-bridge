# Code Of Conduct

## Our Pledge

We want this project to be welcoming, respectful, and useful for everyone.

## Expected Behavior

- Be kind and constructive.
- Assume good intent.
- Keep discussions focused on the project.
- Respect different levels of experience.

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Discriminatory language or behavior
- Sharing private information without permission
- Repeated off-topic disruption

## Enforcement

Project maintainers may remove comments, reject contributions, or block participants who do not follow this code of conduct.
